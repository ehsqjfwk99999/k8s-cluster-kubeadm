k8s-cluster-kubeadm
===================

Setup K8s cluster using kubeadm.

Configuration
-------------
- Node configuration : 1 master + 3 worker
- Provisioning : Vagrant + VirtualBox
- OS : Ubuntu 20.04.3 LTS
- K8s built tool : kubeadm
- CNI : Flannel
